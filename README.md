## Rodando localmente

Antes de começar você precisa do [Git](https://git-scm.com) e [Node](https://nodejs.org/en/) instalado.

```bash

# Clone o repositorio
$ git clone https://github.com/rawallon/alurakut.git && cd alurakut

# Instale as dependencias
$ npm install

# Inicie o servidor
$ npm run dev
```
